BMW E36 3D Showcase

We have put together an interactive 3D showcase for the BMW E36, a true legend in its class. With Three.js, HTML, CSS and JavaScript at its core, this is more than a site with a few still photos; it is an experience.

The site is built to be driven by the scroll. As one makes their way through, the camera will pan and reposition to offer new vantage points on the car, all while the lighting and environment set the scene. You will find HDR effects, some dynamic elements and a night-time feel that make for a compelling view of the E36’s design.
